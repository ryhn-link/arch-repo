ryhn.link Arch Linux repository

Install in a single line

curl https://arch.ryhn.link/install.sh | sudo bash

Or manually :

Append this to /etc/pacman.conf

[ryhn] Server = https://arch.ryhn.link/pkg/$arch

Import and trust the GPG key

gpg --recv-keys 3BC1065444B0109A sudo pacman-key --lsign-key 3BC1065444B0109A
